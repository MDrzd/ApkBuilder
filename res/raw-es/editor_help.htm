Menús Principales:

  • 1) Botón "Guardar": Para guardar los cambios efectuados al archivo.
  • 2) Botón "Copiar": Para copiar al Portapapeles el texto seleccionado.
  • 3) Botón "Pegar": Para pegar el contenido del Portapapeles.
  • 4) Botón "Buscar": Para mostrar u ocultar el Panel de Búsqueda.
  • 5) Botón "Más" (+): Para mostrar el Menú de Opciones emergente.
  • 6) “HTML”: Para mostrar el contenido en formato HTML (sólo lectura).
  • 7) “Paleta de Colores”: Para mostrar el cuadro de diálogo de la Paleta de Colores. Esto le ayudará a obtener un color más adecuado para su uso.
  • 8) “Fgmto. de Código”: Para mostrar el diálogo "Fragmento de Código". Sólo estará operativo cuando se haya abierto un archivo Smali.
  • 9) “Eliminar Líneas”: Para eliminar algunas líneas. Al pulsarlo, aparecerá un cuadro de diálogo en el que se podrá ingresar en los campos “Desde la Línea №” y "Hasta la Línea №” los números de línea deseados para eliminar todo el contenido comprendido entre ellas con un solo clic.
  • 10) “Comentar Líneas”: Para a09adir o eliminar comentarios a ciertas líneas. Sólo estará operativo cuando se haya abierto un archivo Smali.
  • 11) “Ajustes”: Para mostrar los Ajustes del Editor.
  • 12) “Ayuda”: Para mostrar este documento de Ayuda.

    • Panel de Búsqueda:

  • 1) Campo de entrada del Texto a Buscar.
  • 2) Campo de entrada del Texto para Reemplazar.
  • 3) Botón para alternar (sí o no) la función Regex (Expresión Regular).
  • 4) Botón para alternar la Sensibilidad (sí o no) a las Mayúsculas del motor de búsqueda.
  • 5) Campo de entrada para el № de Línea.
  • 6) Botón "Ir": Cuando se establece el № de Línea, le llevará a esa línea. De lo contrario, inserte un carácter '\ n', como si presionara el Botón Ir.
  • 7) Botón "Buscar".
  • 8) Botón "Reemplazar": Reemplaza S07LO la coincidencia encontrada actual.
  • 9) Botón "Reemplazar Todo": Reemplaza TODAS las coincidencias encontradas en el documento actual.
