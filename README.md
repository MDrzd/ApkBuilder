ApkBuilder (Heagoo Modification)

ApkBuilder is a decompiled version of the ApkBuilder app by Heagoo, modified using ApkTool M.

Usage

1. Decompile the APK using ApkTool M


2. Edit Smali files or resources


3. Rebuild the APK


4. Sign the APK
